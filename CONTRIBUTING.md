# Contributing

This is a **showcase repository** for the Intelligent MCP Orchestrator project.

## No Contributions Accepted

This repository is a curated demonstration of the project's capabilities. It is not open for external contributions.

If you're interested in the project:

- **View the source**: [ai-projects-and-management-example](https://github.com/fbratten/ai-projects-and-management-example)
- **Report issues**: Open an issue on the main repository
- **Contact**: jack.rumble556@passfwd.com

## Why No Contributions?

Showcase repositories are intentionally kept simple and curated to:
- Present a clean, focused demonstration
- Avoid complexity from merge conflicts
- Maintain consistent quality and style
